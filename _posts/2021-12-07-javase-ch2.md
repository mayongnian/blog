# java数据类型

## 基本类型

> 1字节（byte） = 8位（bit）

### 整型

| 类型  | 字节数 | 默认值 | 注释       |
| ----- | ------ | ------ | ---------- |
| byte  | 1      | 0      |            |
| short | 2      | 0      |            |
| int   | 4      | 0      |            |
| long  | 8      | 0L     | 末尾加L或l |

### 浮点型

| 类型   | 字节数 | 默认值 | 注释             |
| ------ | ------ | ------ | ---------------- |
| float  | 4      | 0.0f   |                  |
| double | 8      | 0.0d   | 小数默认为double |

### 字符型

| 类型 | 字节数 | 默认值   | 注释               |
| ---- | ------ | -------- | ------------------ |
| char | 2      | 'u\0000' | 本质为无符号位整数 |

### 布尔型

| 类型    | 字节数 | 默认值 | 注释 |
| ------- | ------ | ------ | ---- |
| boolean |        | false  |      |

## 引用类型
